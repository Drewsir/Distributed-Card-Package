## 一、需要启动的服务

先关闭防火墙

```html
service iptables stop
```

### mysql

Windows  会自动启动

### redis

Windows  会自动启动

### hbase

1、启动 zookeeper

```html
zkServer.sh start
zkServer.sh status
```

2、启动 hdfs

```html
sbin/start-dfs.sh
```

3、启动 hbase

```html
bin/start-hbase.sh

bin/hbase shell
```

### kafka

```html
bin/kafka-server-start.sh config/server.properties&
```
